# Research Index

All research notes generated by Revolve, sorted by date (newest first).

```dataview
TABLE
  topic AS "Topic",
  date AS "Date",
  status AS "Status",
  length(sources) AS "Sources"
FROM ""
WHERE type = "research"
SORT date DESC
```

## By Tag

```dataview
TABLE
  topic AS "Topic",
  date AS "Date",
  tags AS "Tags"
FROM ""
WHERE type = "research"
FLATTEN tags
GROUP BY tags
SORT date DESC
```
